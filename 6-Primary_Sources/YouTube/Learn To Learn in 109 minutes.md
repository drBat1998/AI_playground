# Notes
Common learning myth
1. Time
2. IQ
3. Learning style
4. Learning should be easy

Create a learning system.
Information -> ==Encoding== -> ==Retrieval== -> Reencoding

Poor encoding -> poor memory and superficial understanding
First focus on retrieval, then encoding. Encoding upgrading usually needs a lot of time.

==Retrieval== is a using your knowledge. Simulation of real life cases. 
Method, first think about where and how this knowledge would be used.
1. Writing the information.
2. Manipulate the information.
Frequency.
1 day/ 1 weak/ 1 month

Find to do retrieval practice inside your typical workflow. 


Information that comes to your brain should be evaluate and compare to your existence knowledge.
- analogy
- evaluate
- mind map
- comparing
- to teach the information
