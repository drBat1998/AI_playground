# Notes
CellProfiler
- flexible
- scalable
- user friendly
- modular

Cellprofiler workflow
1. Import images
2. Segment objects
3. Population classification
4. Define measurements
5. Export images, figure extra.
![[Screenshot 2025-09-29 at 10.32.21.png]]

Compulsory models
- images - opening images
	- image nomenculature: date_magnification_experiment_condition_.czi
- metadata
	- metadate regex tool - to use information about metadata from the image name
- namesandtypes
	- modules will use this name to refer to these images
Optional modules
- file processing (export, save images)
- image processing (crop, image math, maskimage, smooth, threshold)
- object processing (classify objects, filter objects, identify primary objects)
- measurement
