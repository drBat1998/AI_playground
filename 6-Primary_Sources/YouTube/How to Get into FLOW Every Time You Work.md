# Notes
The flow state reduces consciousness. 

How to activate the Flow state on purpose
- Clear goals
	- Write down goals
	- What it looks like to do it
- Immediate feedback
-  The level of difficulty has to be in the middle of the scale. 
	- If the problem is too difficult, break it into small pieces
- Inreach environment
