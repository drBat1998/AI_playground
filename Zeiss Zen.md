
#### User management
It is turn off by default. 

Groups and users can be assign acordinly. 

Creating a new user
1. Click Tools then Users and Groups
2. In the Users tab click +
3. Enter name, description, and password. 

Creating a new user
1. Click Tools then Users and Groups
2. In the Group tab click +
3. Enter name, description, and password. 

Adding users to a group
1. Click Group
2. Select the group you ant to add a user to
3. Click sun
4. Under Data> Members click +
5. Select the user you want to add to the group and click ok

Managing access rights for user groups
1. Open groups tab
2. Select the group you want to manage access right for
3. Click sun
4. Under Access rights, select the area where you want to restrict acess rights eg

#### Image acquisition
Setting up a new experiment
1. In the left tool area open the acquisition tab.
2. In the experiment manager click sun and select new
3. Enter a name for the experiment.
4. In sun many click save
The experiment would be save within the name, the asterisk after the file name would assign that the settings have been changed without saving. 

Adding channels through Smart Setup
1. On the Acquisition tab, click Smart setup.
2. Select the WF button
3. To add a channel, click +Dyes and contrast method
4. Double click desired dyes
5. Chose between speed, signal and other variatiom.
6. Click set exposure at the top of the acquisition tab

#### Working with Focus strategies
General preparation
1. Activate the acquisition dimensions you want to use for your experiment
2. Open the Focus Strategy tool

Software autofocus
1. First general preparation
2. Select the software autofocus entry. ! Note that the Z value defined by Title setup are ignored. 
3. In Reference Channel section, select the channel that you want to use for the focus action from the list. 
4. Click on the Set as Reference Channel button.
5. In the Time Series Loop and/or Tiles Loop sections of the Focus Strategy tool you can define when focus actions should be performed during the course of the experiment. 
6. Open the Software Autofocus tool 
7. Adjust the autofocus settings to your experiment conditions or use the default settings first. 
8. Set up your tile and/or time series experiment.
9. To start experiment, click on the Start Experiment.

Define Focus in Time Series Experiments
1. First general preparation
2.  Select the Define Focus entry. ! Note that the Z value defined by Title setup are ignored. 
3. In the Stabilization Event Repetitions and Frequency section select Standart mode.
4. Set up a Time Series experiment
5. Use the Live mode to set the focus position using the focus drive.
6. To start experiment, click on the Start Experiment.

Using Local and Global Focus strategy (best for live cell imaging)
1. Select the Use Z-value/focus surface define in tiles setup.
2. If you want to set up the entire focus strategy, you can click optimize this focus strategy to to use focus strategy wizard.
3. In Zvalue/focus surface section, select Local
4. Under Initial Definition for Z-values/Focus surface, select the by Tiles Setup
5. Set up a Tiles experiment
6. Create alocal focus surface for the tile regions
7. To start experiment, click on the Start Experiment.
