
# Включення конфокального мікроскопа
1. Вімкнути компютер
2. Вімкнути NRAF Power-Manager![[IMG_0575.png]]
3. Вімкути лазери 543 nm та 633 nm![[IMG_0576.png]]
4. Вімкнути Laser почекати та перевести ключ в позицію On![[IMG_0577.png]]
5. На компютері увійти в FV10 - Olympus FLUOVIEW

# Настройка поля зору
![[IMG_0579.png]]
1. Size 340 by 340
2. Microscope  - вибрати збільшення на якому диветесь
3. В Image Acqusition control включит світло на вибрати поле зору
4. Виключити світло
5. Вімкнути лазери та TD1(фазовий контраст)
6. Вімкнувши XY Repeat по черзі настроїти HV та Offset, не використовуючи Gain.

# Отримання фотографій
1. Зупинити XY Repeat натиснувши Stop
2. Перевести Size 1000 by 1000
3. Натиснути XY для та почекати завершення сканування
4. File -> Export
5. Вибрати шлях та назву папки та переконатись що поля згадані нище заповненні.
![[IMG_0585.png]]
File type: TIFF
Bit Depth 24
Output Format RGB color
and Merge Channel
