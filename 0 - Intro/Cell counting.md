# cropping picture
1. Drug a package of pictures to ImageJ 
2. Select a well-known distance or control measure by the straight tool.
3. Add um value if needed: Analyse -> Set Scale -> ...
4. Specify location: Edit ->  Selection -> Specify -> scaled units (um)
5. Image -> Crop
6. Repeat 1-6 steps to crop all of the images.
